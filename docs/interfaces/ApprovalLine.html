    Usage and Relationships

    The ApprovalLine represents a step in an approval process. It is often linked to approval processes and users to manage workflows.

    Related Types:

    • ApprovalProcess: The process to which this approval line belongs.
    • User: The user responsible for this approval step.

    Key Fields:

    • id: Unique identifier for the approval line.
    • status: Indicates the current state of the approval (e.g., pending, approved).
    • sequence: The order of this step in the approval process.
